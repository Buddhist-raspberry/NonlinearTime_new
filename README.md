# NonlinearTime

## 开发环境

* 开发引擎：Unity 2020.2.3.f1

---

## 项目结构

* `Assets`
  * `Animation`：动画相关，Animation和Animator Controller
  * `Ludiq`：Chorons插件
  * `Materials`：材质和shader
  * `Models`：模型
  * `Packages`：第三方Packages
  * `Plugins`：DOTween插件
  * `Prefabs`：预制体
  * `Resources`：DOTween设置文件
  * `Scenes`：场景
  * `Scripts`：C#脚本
  * `Settings`：光照设置文件
  * `Sprites`：2D图片
